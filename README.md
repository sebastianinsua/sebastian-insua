```markdown
# sebastian-insua — Sitio personal de Sebastián Insúa

Plantilla one-page para consultor industrial. Incluye HTML, CSS y JS minimalistas.

Pasos rápidos para usar:
1. Reemplaza textos si lo deseas en `index.html` (nombre, email, enlaces).
2. Configura el formulario:
   - Regístrate en Formspree y sustituye la acción del form en `index.html` por `https://formspree.io/f/TU_ID`.
   - O cambia por tu endpoint de correo o webhook.
3. Despliegue:
   - GitHub Pages: el sitio estará disponible en `https://sebastianinsua.github.io/sebastian-insua` una vez que los archivos estén en la rama `main`.
   - Netlify/Vercel: conecta el repo o arrastra la carpeta para desplegar.
4. Opcional: añade Google Analytics, meta tags Open Graph, o imágenes en la carpeta `assets/`.

Archivos:
- `index.html` — página principal
- `styles.css` — estilos
- `scripts.js` — JS mínimo (menú móvil y formulario)

Si querés, puedo ayudarte a ajustar colores, textos o publicar en Pages después de que confirmes que el push se hizo correctamente.
```
